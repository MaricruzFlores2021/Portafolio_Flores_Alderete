FUNDAMENTOS DE LA TECNOLOGÍA WEB



En el desarrollo de software, existen dos enfoques principales: la Tecnología Web, aplicaciones basadas en navegadores, y la Tecnología de Software Nativo, aplicaciones específicas para plataformas; su elección depende de las necesidades del proyecto.

• CONTEXTO: En el contexto de las tecnologías web, HTML organiza contenido, CSS controla la apariencia y HTTP transmite datos en la web.

• JAVASCRIPT: En el desarrollo web, JavaScript permite la creación de sitios web interactivos en el navegador. Las Web APIs son utilizadas para diversas tareas, como manipular el DOM, reproducir medios y gestionar eventos en páginas web y aplicaciones.

• FUNCIÓN: El frontend es lo que ven los usuarios e incluye elementos visuales, y permite a los a estos interactuar con la aplicación. El back-end son los datos y la infraestructura que permiten que la aplicación funcione.

• ROLES: Los roles en proyectos de software abarcan gestores de proyecto, arquitectos de información, diseñadores gráficos, especialistas en usabilidad y experiencia de usuario, administradores de bases de datos, redactores de contenidos y especialistas en seguridad de la información.

• UX AND UI DESIGN: El diseño UX/UI busca crear experiencias agradables y eficientes para los usuarios al interactuar con productos y servicios, utilizando herramientas como Figma y Adobe XD para la interfaz de usuario. El objetivo es que la experiencia sea satisfactoria y atractiva.

• PROCEDIMIENTOS: Durante la sesión, nos dedicamos a llevar a cabo un exhaustivo proceso de revisión de la bibliografía pertinente al tema que estamos abordando. Este procedimiento implicó una evaluación minuciosa de fuentes académicas y literarias relevantes, con el objetivo de obtener una comprensión más profunda y detallada de los aspectos clave que rodean nuestro tema de estudio.

• RESULTADOS: Se ha alcanzado un entendimiento sólido en los principios fundamentales de la tecnología web, consolidando conocimientos profundos en áreas clave como HTML, CSS y JavaScript. Además, se ha desarrollado una comprensión integral de protocolos web, como HTTP, y se ha aplicado de manera práctica el Modelo de Objetos del Documento (DOM) y las API web, proporcionando una base firme para abordar proyectos web con confianza y eficiencia.
