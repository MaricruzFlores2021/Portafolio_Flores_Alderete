Conclusión


En estas enriquecedoras ocho semanas inmerso en el fascinante universo de la tecnología web, he consolidado no solo conocimientos fundamentales, sino también habilidades avanzadas que me otorgan la confianza necesaria para abordar proyectos web con eficiencia y destreza. Desde el inicio, al comprender el papel esencial de HTML y su integración en el Modelo de Objetos del Documento, sentí cómo se establecían los cimientos sólidos que han impulsado mi progreso en el desarrollo web.
A medida que avanzaban las semanas, mi horizonte se expandió hacia CSS, diseño responsivo y JavaScript, explorando la creación de elementos interactivos y gráficos a través del cautivador lienzo de HTML Canvas. La transición hacia React representó un hito trascendental, transitando desde el desarrollo basado en componentes hasta la habilidosa gestión dinámica de interfaces mediante props y la incorporación de Hooks para manejar el estado y los efectos de manera más eficiente.
Además, al reflexionar sobre el conjunto de conocimientos adquiridos, me maravillé al reconocer la magnitud de mi progreso. Desde la concepción de interfaces estáticas hasta la implementación de estrategias avanzadas como la creación de Hooks personalizados en React, me encuentro listo y entusiasmado para enfrentar desafíos aún más complejos en el desarrollo web. Este viaje de aprendizaje continuo y crecimiento me inspira, y estoy emocionado por el camino que se extiende ante mí, lleno de nuevas oportunidades y descubrimientos en la vastedad del desarrollo web.
En las últimas semanas de aprendizaje del segundo consolidado, se exploraron tecnologías clave en desarrollo web, desde la agilidad de Next.js hasta la robustez de Java y Spring en el backend. La combinación de estrategias como CSR/SSR, TypeScript, y la implementación de API Rest evidencian una evolución hacia prácticas más sólidas. La experiencia práctica con desafíos como las diferentes API y la creación de sistemas backend eficientes con Eureka y Spring demuestran la aplicabilidad de estos conocimientos en proyectos reales. Finalmente, la innovación y practicidad convergen para crear aplicaciones web modernas y eficaces.



Reflexión y Bibliografía (Semana 1-16)

Reflexión por semana

Semana 01

Esta semana, consolidé conocimientos clave en tecnología web, desde HTML hasta diseño UX/UI. Comprender los roles en proyectos y aplicar el Modelo de Objetos del Documento me brindó confianza para enfrentar proyectos web eficientemente. Lista para futuros desafíos con una base sólida.

Semana 02

La semana de HTML amplió mi comprensión de su papel esencial en la construcción web, desde su definición y etiquetas clave hasta la práctica en Visual Studio Code. El enfoque en el DOM y el Critical Rendering Path resaltó su importancia para una visualización eficiente.

Semana 03

Esta semana, al explorar CSS y Diseño Responsivo, comprendí la importancia crítica de las hojas de estilo en la presentación web. Aprendí técnicas precisas de aplicar estilos, y las herramientas como Flexbox y Grid transformaron mi enfoque en el diseño eficiente. La práctica en Visual Studio Code consolidó estos conocimientos.

Semana 04

Esta semana, profundicé en JavaScript y HTML Canvas, comprendiendo variables, operadores, estructuras de control y objetos. Experimenté con la práctica en Visual Studio Code, creando un reloj analógico y un gráfico circular con Canvas HTML, fortaleciendo mis habilidades de desarrollo front-end.

Semana 05

Me sumergí en el desarrollo basado en componentes con JSX y abordé técnicas de estilización, desde estilos inline hasta Styled Components. Seguí un procedimiento que incluyó la instalación de Node.js y la creación de proyectos React con opciones como Create React App y Next.js, consolidando mis habilidades para proyectos más avanzados.

Semana 06

Aprendí a transmitir información en React mediante props y children, consolidando habilidades esenciales para construir interfaces dinámicas y eficientes en aplicaciones web.

Semana 07

Se profundizó en los Hooks de React, herramientas poderosas para gestionar estado y efectos en componentes de función. Destacó casos clave, como useState para el estado dinámico y useEffect para acciones secundarias. La creación de Hooks personalizados fue resaltada como estrategia para mejorar la reutilización del código, simplificando y optimizando el proceso de desarrollo en React.

Semana 08

Se evaluó todos los conociemtos adquiridos mediante la teória y práctica de los temas expuestos en el primer consolidado.

Semana 09

Next.js, con CSR/SSR, TypeScript y estilos modulares, brinda una base ágil y robusta. La eficiente gestión de formularios y la migración a TypeScript mejoran la calidad del código, reflejando una evolución hacia prácticas más sólidas en el desarrollo web.

Semana 10

Next.js destacó que estrategias de generación estática como Static Generate y Static Regeneration, optimiza la regeneración y actualización incremental. El reto de la API de Marvel ofreció experiencia práctica, consolidando a Next.js como herramienta eficaz en desarrollo web moderno.

Semana 11

La combinación de getServerSideProps en Next.js y Next Auth simplifica el desarrollo web. Con pasos claros, desde la instalación hasta la personalización, se logró implementar eficientemente un subsistema de autenticación en Next.js, demostrando la potencia y accesibilidad de estas herramientas.

Semana 12

La semana de desarrollo backend brindó conocimientos esenciales sobre arquitectura web y el uso efectivo de Java y Spring. La instalación de herramientas como IntelliJ Idea y Spring Boot facilitó el avance significativo en el desarrollo del sitio web backend de productos.

Semana 13

El desarrollo de una API Rest con Spring implica comprender principios como HTTP y MVC, desde la instalación de OpenJDK hasta la creación de modelos y servicios. Los resultados muestran un progreso destacado con una API funcional para gestionar cursos en el backend del sitio web.

Semana 14

La sesión enfocada en el desarrollo de una API REST con Spring resaltó la importancia de la modularidad y comprensión profunda de HTTP. La configuración precisa y el uso de herramientas como Eureka y Spring simplifican la creación de sistemas backend eficientes. La práctica directa brinda experiencia valiosa para proyectos futuros.

Semana 15

Se consiguió una aplicación web completa con elegante diseño en NEXT JS y backend eficiente en Spring y MySQL. Se fusiona experiencia de usuario y sólida arquitectura, mostrando que la innovación y practicidad pueden coexistir.

Semana 16

Se evaluó todos los conociemtos adquiridos mediante la teória y práctica de los temas expuestos en el segundo consolidado.
