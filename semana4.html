Desarrollo Fronted, JS y Canvas


En esta sección se abarca diversos aspectos de JavaScript y HTML Canvas. Se explora la creación y manipulación de variables, operadores, estructuras de control, funciones, módulos, objetos y clases en JavaScript. También se detalla el uso de Canvas para la creación de gráficos.


DESARROLLO FRONTED – JAVASCRIPT - CANVAS



• VARIABLES Y OPERADORES: En JavaScript, las variables son espacios de memoria con valores y nombres, pudiendo ser globales, locales o constantes. Los tipos de datos incluyen booleanos, null, undefined, números, BigInt, cadenas, símbolos y objetos.
o VARIABLES:Las variables globales se asocian con el objeto global (window en páginas web). Se puede usar undefined para verificar si una variable tiene valor, y este se comporta como falso en contextos booleanos. El hoisting permite referenciar variables declaradas posteriormente, devolviendo undefined si no están inicializadas.
o OPERADORES:Los operadores incluyen aritméticos, lógicos, de comparación, asignación, condicionales, unarios, delete, typeof, y void. Además, el operador spread expande elementos iterables como arrays, cadenas, objetos, NodeList y más.

• ESTRUCTURAS DE CONTROL: Se exploran los condicionales if y switch, destacando valores verdaderos y falsos. En cuanto a las Estructuras Repetitivas o Bucles, se presentan diversas construcciones, como for, while, do...while, forEach, for/of, y for/in, proporcionando herramientas variadas para el control del flujo de ejecución y la repetición en programas JavaScript.

• FUNCIONES Y PROCEDIMIENTOS: El panorama de funciones en JavaScript abarca las tradicionales, definidas con la palabra clave "function"; las anónimas, sin un nombre específico y comúnmente asignadas a variables; las funciones construidas con el constructor Function(); las autoinvocadas, que se ejecutan automáticamente tras su definición; las de flecha, para expresiones más concisas; el objeto Arguments, que almacena los argumentos de una función; y las funciones closures, que contienen otras funciones y acceden a su ámbito, preservando la privacidad de variables.

• CADENAS: En JavaScript, las cadenas representan texto y se crean con literales simples o dobles, o mediante objetos de cadena usando el constructor String(). Los literales pueden incluir secuencias de escape para caracteres especiales. Las cadenas, instancias de la clase String, ofrecen métodos como charAt, charCodeAt, indexOf, concat, slice, etc., que facilitan operaciones como extracción de subcadenas y búsqueda de índices.

• ARRAYS, NUMBER Y MODULOS: Los arrays son colecciones versátiles que pueden contener diversos datos. Se declaran y acceden mediante corchetes, y ofrecen métodos como sort, pop, push, reverse, join, map, filter, slice, etc. Los números, representados por el tipo de datos Number, pueden declararse de forma primitiva o mediante la clase Number(). Funciones útiles como parseInt, parseFloat, toFixed, y toExponential simplifican el trabajo con números. El objeto Math proporciona funciones matemáticas esenciales, como Math.PI, Math.E, Math.random(), Math.pow, y Math.sqrt, facilitando operaciones comunes.

• OBJETOS Y CLASES: Los objetos son colecciones de pares nombre-valor, creados con Object() o notación de llaves. Se accede a sus propiedades con notación de punto o array. La desestructuración de objetos facilita asignar valores a variables. Las clases, introducidas con "class," permiten la creación de instancias y herencia de propiedades y métodos.

• CANVAS: HTML Canvas es un elemento fundamental para la creación de gráficos en JavaScript, actuando como un lienzo que se puede manipular mediante scripts para dibujar elementos como rectángulos, círculos y texto. Utiliza un sistema de coordenadas bidimensional, y sus métodos, como arc y fillRect, permiten realizar diversas operaciones gráficas. Además, se pueden modificar propiedades como el grosor de línea y trabajar con texto, estableciendo fuente y posición.

• PROCEDIMIENTOS: Durante la sesión, nos dedicamos a llevar a cabo un exhaustivo proceso de revisión de la bibliografía pertinente al tema que estamos abordando. Además, en Visual Studio Code practicamos Canvas HTML.
